Aula 01 - Cores

Existem várias formas de definirmos cores utilizando o CSS. Elas são:

  • Cores pré-definidas
  • Palavra-chave "currentcolor"
  • Hexadecimal
  • Hexadecimal com transparência
  • RGB
  • RGBA (com transparência)
  • HSL
  • HSLA (com transparência)

HSL

HSL (hue, saturation, lightness):

  • Hue (matiz): é o grau na roda de cores (de 0 a 360 graus)
    • 0 ou 360 é vermelho
    • 120 é verde
    • 250 é azul
  • Saturation (saturação): valor percentual
    • 0% um tom de certeza
    • 100% cor total
  • Lightness (luminosidade): luminosidade da cor
    • 0% é preto
    • 100% é branco

Bordas com Imagens

  • border-image-source - fonte da imagem
  • border-image-width - largura das bordas (funciona como o width normal)
  • border-image-repeat - repetição da imagem
  • border-image-outset - o quanto a imagem vai ficar para fora do elemento.
  • border-image-slice - fatiar a imagem ou não dos lados

Há a possibilidade de utilizar somente uma vez a 'border-image' e atribuir todos os atributos de uma vez apenas.

Caso você não passe uma das propriedades, ela vai ser definida como um padrão.

Propriedade 'font-weight'

Basicamente, altera a espessura do texto.

  • font-weight: lighter
  • font-weight: normal
  • font-weight: bolder

Lembrando que também funciona com núemro em pixels.

Propriedade 'font-stretch'

Esta propriedade só funciona se a fonte que está sendo utilizada tem a propriedade.

Valores para exemplo (Não está sendo aplicado no estilo):

  • font-stretch: ultra-condensed;
  • font-stretch: extra-condensed;
  • font-stretch: normal;
  • font-stretch: expanded;
  • font-stretch: ultra-expanded;

Propriedade 'line-height'

Basicamente, trata-se do espaçamento, (altura). Valor do exemplo em ' 1.5'. Sempre em relação ao tamanho do texto, o recomendado é sempre usar mínimo com 1.5.

Propriedade 'word-wrap'

Define, caso uma palavra seja muito grande para caber em uma linha, se ela será quebrada em qualquer caractere para que uma parte dela fique e o restante na próxima, ou se não, toda palavra deve ficar na mesma linha e ultrapassar os limites que definimos para o elemento (uma div, por exemplo).

  • word-wrap: normal;
  • word-wrap: break-word;

Propriedade 'word-break'

Define quando o nosso texto deve ter uma quebra de linha. Exemplos de valores:

  • word-break: normal;
  • word-break: break-all;
  • word-break: keep-all;

Propriedade 'writing-mode'

Define a orientação do texto (vertical ou horizontal). Exemplos de valores:

  • writing-mode: horizontal-tb;
  • writing-mode: vertical-rl;
  • writing-mode: vertical-lr;
